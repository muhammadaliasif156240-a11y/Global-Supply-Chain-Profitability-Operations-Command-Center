# Global-Supply-Chain-Profitability-Operations-Command-Center
End-to End Global Supply Chain &amp; Profitability Operations to identify various patterns and aspects using  Microsoft Excel, MYSQL, Microsoft Power BI.
**Tech Stack**: MySQL, Power BI, Advanced Excel (Power Query), DAX
**Key Engineering & Analytics**:
**Data Pipeline & ETL**: Extracted and cleaned a massive corporate logistics dataset using Power Query, handling missing values and standardizing date formats for time-series intelligence.
**Database Management (SQL)**: Engineered a relational database in MySQL. Wrote complex queries to aggregate critical business metrics, including total financial loss due to suspected fraud and department-level order processing loads.
**Advanced UI/UX Design**: Designed a custom "Executive Boardroom" interface using Canva (Glassmorphism design) to provide a premium, distraction-free user experience for corporate stakeholders.
**Dynamic Visualization (Power BI**):
Built predictive forecasting models to project future sales and order volumes.
Programmed custom DAX measures to calculate dynamic metrics like Fraud Risk Capital ($826K) and Late Delivery Rates.
Engineered interactive Report Page Tooltips to provide deep-dive insights (e.g., identifying "Standard Class" as the primary driver of shipping delays) without cluttering the main dashboard.
**Business Impact**:
Provided a centralized ERP dashboard that allows the CFO to track financial leakage, the COO to optimize shipping routes, and the CEO to monitor global market profitability at a single glance.
